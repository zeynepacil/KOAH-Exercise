# KOAH Exercise Mobile Application

## 📌 Proje Hakkında

Bu proje, KOAH (Kronik Obstrüktif Akciğer Hastalığı) tanılı bireyler için geliştirilmiş bir mobil egzersiz uygulamasıdır.  
Uygulama, solunum rehabilitasyonu kapsamında önerilen nefes ve güçlendirme egzersizlerini kullanıcıya mobil ortamda sunmayı amaçlamaktadır.

Proje React Native ve Expo kullanılarak geliştirilmiştir.

---

## 🚀 Kullanılan Teknolojiler

- React Native
- Expo
- JavaScript
- EAS Build (Android APK oluşturma)

---

## 📱 Uygulama Özellikleri

- Bilgilendirme ve Onam Formu Ekranı
- Isınma Hareketleri Ekranı
- Nefes Egzersizi Ekranı
- Egzersiz Başlat Butonu ve Geri Sayım Mekanizması
- Günlük İlerleme Görüntüleme
- Yönetim Paneli (Video, Blog, Kullanıcı Ekleme)
- Profil ve Bildirim Bölümü
